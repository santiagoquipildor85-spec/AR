📖 Instrucciones de Uso

  1. Generar código: Selecciona un número del 1-99 y escribe tu mensaje
  2. Descargar: Haz clic en "Descargar Imagen" para guardar el código
  3. Imprimir: Imprime el código en papel o muéstralo en pantalla
  4. Escanear: Usa la aplicación AR para detectar el código
  5. Configurar: Si usas un código nuevo, actualiza ar-barcode.html
